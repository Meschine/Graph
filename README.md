# График функции
Поддерживаемые функции:
+ sin(x) - синус x
+ cos(x) - косинус x
+ tan(x) - тангенс x
+ asin(x)	- арксинус x
+ acos(x)	- арккосинус x
+ atan(x)	- арктангенс x
+ sqrt(x) - квадратный корень x
+ log(x) - натуральный логарифм x
+ abs(x) - абсолютное значение x
+ exp(x) - e^x
